# SoftMatterGPT 演示流程

本文档按页面顺序描述 SoftMatterGPT 的完整演示路径，包含每步操作和预期结果。演示前请确保后端服务和前端页面均已启动。

---

## 演示准备

1. 启动后端服务：`python api_server.py`
2. 启动前端页面：`streamlit run app.py`
3. 在浏览器中打开 `http://localhost:8501`
4. 如需 LLM 功能演示，确保 Ollama 已启动并拉取了至少一个模型（如 `qwen2.5:7b`）

---

## 第一步：首页总览

**页面**：首页

**操作**：打开系统，浏览首页内容。

**预期结果**：
- 显示 SoftMatterGPT 项目名称和定位描述
- 状态卡片展示：当前模型名称、Ollama 连接状态（在线/离线）、课程知识库 60 条、文献记录 27 条、实验配方 30 条
- 显示今日提问次数和累计问答次数
- 展示系统架构流程图（用户浏览器 -> Streamlit -> FastAPI -> 核心模块 -> 数据层 -> Ollama）
- 页面底部显示安全声明

**讲解要点**：介绍系统整体架构和各模块关系，说明数据规模为教学示例。

---

## 第二步：系统设置

**页面**：系统设置

**操作**：
1. 点击"检测 Ollama 连接"按钮
2. 点击"刷新本地模型列表"
3. 在模型下拉框中选择一个模型（如 qwen2.5:7b）
4. 点击"设为当前模型"
5. 点击"测试当前模型"

**预期结果**：
- 连接检测显示 Ollama 在线或离线状态
- 模型列表展示本地已拉取的模型（或离线时显示空列表提示）
- 当前模型设置成功
- 模型测试返回成功或离线时显示友好提示

**讲解要点**：演示系统对 Ollama 的连接管理和模型切换能力，以及离线时的优雅降级。

---

## 第三步：软物质智能问答

**页面**：软物质智能问答

**操作**：
1. 在输入框中输入问题：`什么是剪切变稀？为什么洗发水和番茄酱会表现出类似行为？`
2. 点击"提问"按钮
3. 浏览回答内容和证据卡片
4. 点击"满意"反馈按钮

**预期结果**：
- 系统从知识库、文献库、配方库检索相关证据
- 回答包含：简要结论、详细解释、例子说明、容易混淆点、参考证据、学习建议
- 证据卡片展示 Top-K 条证据，包含标题、分类、来源、相似度、内容摘要
- 显示置信度标签（高/低）
- 反馈保存成功

**如果 Ollama 离线**：证据检索正常展示，LLM 回答区域显示错误提示，不会崩溃。

**讲解要点**：重点演示 RAG 流程——先检索再生成，展示证据来源和置信度。

---

## 第四步：文献知识检索

**页面**：文献知识检索

**操作**：
1. 在搜索框输入关键词：`水凝胶`
2. 浏览检索结果
3. 尝试按年份或材料体系筛选
4. 点击一条文献记录查看详情

**预期结果**：
- 返回匹配的文献记录列表
- 每条记录展示：论文标题、年份、来源、材料体系、实验方法、表征方式、结果摘要
- 筛选功能正常工作
- 数据来源标注为教学示例

**讲解要点**：演示结构化文献记录的检索能力和多维筛选。

---

## 第五步：实验配方库

**页面**：实验配方库

**操作**：
1. 在搜索框输入关键词：`PVA`
2. 浏览配方检索结果
3. 查看配方的安全等级标签
4. 观察高风险配方的教师复核提示

**预期结果**：
- 返回匹配的实验配方列表
- 每条配方展示：材料、浓度、工艺条件、温度、时间、表征结果
- 安全等级标签（高/中/低）正常显示
- 高风险配方显示教师复核提示
- 教学示例数据有明确标注

**讲解要点**：演示配方检索和安全审核机制。

---

## 第六步：实验方案助手

**页面**：实验方案助手

**操作**：
1. 在研究目标输入框中输入：`制备一种具有剪切变稀特性的水凝胶，用于 3D 打印`
2. 点击"生成实验方案"按钮
3. 浏览生成结果

**预期结果**：
- 生成的教学版实验方案包含：实验目标、设计思路、参考配方、材料体系、变量设计、表征方法、预期现象、风险提醒、教师复核清单
- 输出明确标注为"教学版实验方案框架"
- 高风险试剂和条件自动标记教师审核
- 显示安全免责声明

**如果 Ollama 离线**：接口正常返回，显示 LLM 不可用的友好提示。

**讲解要点**：强调"教学版"定位和安全审核机制。

---

## 第七步：表征结果分析

**页面**：表征结果分析

**操作**：
1. 在文字描述框中输入：`流变测试显示剪切速率增大时粘度明显下降，表现出明显的剪切变稀行为`
2. 点击"分析"按钮
3. 浏览分析结果

**预期结果**：
- 分析输出包含：观察到的现象、可能对应结构、可能工艺原因、建议补充表征、风险与不确定性
- 匹配剪切变稀相关规则，置信度为中
- 结合知识库证据给出解释
- 显示免责声明："图像和数据分析结果仅作为教学解释，不作为最终实验结论"

**讲解要点**：演示基于规则匹配的表征分析能力，说明当前为 MVP 阶段。

---

## 第八步：材料体系对比

**页面**：材料体系对比

**操作**：
1. 选择一个预设对比主题（如"物理交联水凝胶 vs 化学交联水凝胶"）
2. 点击"对比"按钮
3. 浏览对比结果

**预期结果**：
- 生成 Markdown 格式的对比表格
- 展示各项性能指标的对比结果
- 对比结论附带证据来源

**如果 Ollama 离线**：接口正常返回，显示 LLM 不可用提示。

**讲解要点**：演示材料体系的结构化对比能力。

---

## 第九步：教师分析面板

**页面**：教师分析面板

**操作**：进入页面后直接浏览各项统计内容。

**预期结果**：
- 最近提问记录列表
- 高频关键词柱状图：展示学生高频关注的知识点（如水凝胶、表面活性剂、胶体等）
- 反馈分布饼图：展示正面、中性、不满意反馈的占比
- 每日提问趋势折线图：展示提问数量的时间分布
- 弱点总结：基于低置信度问答分析学生薄弱知识点
- 课堂复习建议：根据高频问题和薄弱点自动生成

**讲解要点**：演示教师端的数据分析能力，说明如何辅助教学改进。

---

## 补充演示（可选）

### 幻觉防护演示

在智能问答页面输入一个知识库中不存在的问题，如一个虚构的材料名称，观察系统是否能识别证据不足并拒绝编造回答。

### 安全审核演示

在实验方案助手中输入包含高风险试剂的研究目标（如涉及浓硫酸），观察系统是否自动标记高风险并生成教师复核清单。

### API 接口演示

使用 curl 或浏览器直接访问 `http://127.0.0.1:8000/docs`，展示 FastAPI 自动生成的 API 文档页面。
