# SoftMatterGPT — AI 教学工作台

面向软物质课程与实验教学的智能辅助系统，基于 RAG 检索增强生成技术，让每一个回答都有据可依。

## 项目简介

SoftMatterGPT 是一个专为软物质科学课程设计的 AI 教学工作台。系统整合了知识问答、文献检索、实验配方查询、表征分析等功能，帮助学生深入理解软物质科学的核心概念，辅助教师进行教学分析。

## 核心功能

### 1. 智能问答
- 基于 RAG（检索增强生成）技术回答课程概念问题
- 展示证据来源，确保回答可追溯
- 支持置信度评估和用户反馈

### 2. 文献检索
- 27 篇结构化文献记录，覆盖流变学、自组装、乳液、水凝胶、表面活性剂、表征方法等领域
- 支持关键词、作者、年份、知识领域多维筛选
- 点击查看详情，包含摘要、关键词和材料体系信息

### 3. 实验配方
- 30 条实验配方，涵盖水凝胶制备、表面活性剂配制、纳米粒子合成、乳液制备等
- 安全等级标注（低/中/高风险）
- 详细的实验步骤、温度条件、反应时间和预期结果

### 4. 实验方案生成
- 输入研究目标，AI 生成教学版实验方案框架
- 包含实验材料、步骤、预期结果和安全提示

### 5. 表征分析
- 18 条分析规则，覆盖流变学、DLS、XRD、SEM/TEM、DSC、GPC、NMR 等表征方法
- 支持关键词和表征方法搜索
- 点击查看详细解释和应用场景

### 6. 材料对比
- 预设对比主题和自定义对比功能
- 结构化对比表格，展示不同材料体系的性能差异

### 7. 教师分析
- 学生问答统计（总问答、本周新增、满意率、活跃学生）
- 每日趋势柱状图和知识领域分布图
- 最近问答记录和置信度分析
- 知识薄弱点分析和教学建议

### 8. 系统设置
- Ollama 服务连接配置和状态检测
- 模型管理（选择、设为当前、测试）
- 快速测试模型响应
- 技术栈信息展示

## 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | Next.js 15 + React 19 |
| UI 框架 | Tailwind CSS 4 |
| 类型系统 | TypeScript |
| 后端服务 | FastAPI |
| 大语言模型 | Ollama |
| 检索技术 | TF-IDF + 余弦相似度 |
| 数据存储 | CSV 文件 |

## 设计风格

采用 **科技玻璃风（Tech Glass）** 设计语言：

- **星空背景**：深空渐变底色 + CSS 星点 + 星云光晕
- **玻璃拟态**：半透明毛玻璃卡片，backdrop-filter 模糊效果
- **化学元素装饰**：苯环、分子、试管、烧瓶、DNA 螺旋、原子等 SVG 图标
- **多彩渐变文字**：紫色、青色、粉色、白色渐变标题
- **可收缩侧边栏**：70px 收缩态，hover 展开至 200px

## 快速开始

### 环境要求

- Node.js 18+
- npm 或 yarn

### 安装

```bash
# 克隆项目
git clone https://github.com/yourusername/softmatter-gpt.git
cd softmatter-gpt/frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 查看效果。

### 构建

```bash
npm run build
npm start
```

## 使用说明

1. **首页**：浏览系统功能概览和技术栈信息
2. **问答**：输入软物质相关问题，获取 AI 回答和证据来源
3. **文献**：搜索和浏览结构化文献记录，点击查看详情
4. **配方**：查询实验配方，查看安全等级和实验步骤
5. **方案**：输入研究目标，生成实验方案框架
6. **表征**：搜索表征分析规则，查看详细解释
7. **对比**：对比不同材料体系的性能差异
8. **数据**：查看学生问答统计和教学建议
9. **设置**：配置 Ollama 服务和模型

## 安全声明

本系统输出的实验方案仅为教学参考框架，不构成可直接执行的实验操作 SOP。具体试剂选择、浓度、温度、时间和操作条件必须经教师或实验室安全规范审核后方可实施。
